Frontend Mentor challenge

Frontend Mentor simulates being given a style guide and existing designs to work with.

This is useful for thinking about situations where assets or design decsions are already in place.

I picked this profile card exercise as my first challenge as I wanted to try out using either CSS flex or grid.

I learnt that flex box is the best option when you only have a single element to position.

With CSS grid the card only centered horizontally and not vertically, so I needed to add in a top margin.

I used negative margins to position the larger blue circles as background images.

I've since learnt of other options for positioning, including the ::before and ::after pseudo properties
